// STRYKE-NEO4J — NEO4J GRAPH CLIENT FOR STRYKE

Engineering report GitHub strykelang Meta

What it is

stryke-neo4j is a Neo4j graph database client for stryke: parametrized Cypher query and run, scalar/row helpers, and schema introspection over the Bolt protocol — via neo4rs, the pure-Rust driver. No JVM. The cdylib owns one tokio runtime and presents a blocking facade, caching a connection pool per (uri, user, db).

Quick start

use Neo4j

var %conn = ( uri => "neo4j://localhost:7687", user => "neo4j", password => $ENV{NEO4J_PASS} )

Neo4j::run("CREATE (p:Person {id: $id, name: $name})", params => { id => 1, name => "Ada" }, %conn)

val @people = Neo4j::query("MATCH (p:Person) WHERE p.id = $id RETURN p.name AS name",
                           params => { id => 1 }, %conn)
p $people[0]{name}        # Ada

p Neo4j::scalar("MATCH (p:Person) RETURN count(p) AS n", %conn)

Rows as JSON

neo4rs deserializes each Bolt row straight into JSON, so a returned node or relationship becomes its property map and arbitrary results round-trip without a per-query schema. Parametrized Cypher binds $name placeholders from params, so values never enter the query text.
